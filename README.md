## 项目名称
CMS 服务器

## 项目描述
这是一个内容管理系统（CMS）服务器项目，旨在提供高效、可靠的内容管理和分发服务。该项目支持多种内容类型，并提供灵活的API接口，方便用户进行内容的创建、更新、删除和查询操作。

## 功能特性
- 内容创建、读取、更新和删除（CRUD）操作
- 用户认证和授权
- RESTful API 接口
- MySQL数据库支持

## 技术栈
- 编程语言：Rust
- 后端框架：axum
- 数据库：MySQL
- 认证：JWT（JSON Web Token）

## 安装和使用
```
cargo run --release
```
